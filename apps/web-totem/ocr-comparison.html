📝 Como configurar APIs no backend

1. Edite apps/api/.env

2. Configure as credenciais:

# Google Cloud Vision (Service Account)
GOOGLE_APPLICATION_CREDENTIALS=./totemcacim-xxxxx.json

# AWS Textract
AWS_REGION=sa-east-1
AWS_ACCESS_KEY_ID=...
AWS_SECRET_ACCESS_KEY=...

# Azure Computer Vision
AZURE_VISION_API_KEY=...
AZURE_VISION_ENDPOINT=https://...

3. Reinicie o backend: npm run start:dev

🏆 Vencedor

📚 Como obter as APIs

🌐 Google Cloud Vision:

  1. Acesse: console.cloud.google.com
  2. Crie um projeto → APIs & Services → Enable APIs → Vision AI
  3. Credentials → Create Credentials → API Key
  4. Copie a chave e cole acima

💰 Gratuito: 1.000 requisições/mês

☁️ AWS Textract:

  1. Acesse: aws.amazon.com/console
  2. IAM → Users → Create User → Attach Policy: AmazonTextractFullAccess
  3. Security Credentials → Create Access Key
  4. Copie Access Key ID e Secret Access Key

💰 Gratuito: 1.000 páginas/mês (3 primeiros meses)

🔷 Azure Computer Vision:

  1. Acesse: portal.azure.com
  2. Create Resource → AI + Machine Learning → Computer Vision
  3. Keys and Endpoint → Copie Key 1 e Endpoint

💰 Gratuito: 5.000 transações/mês

🔤 Tesseract.js:

✅ Gratuito e local - sem necessidade de API!
